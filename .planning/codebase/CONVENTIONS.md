# Coding Conventions

## Dart / Flutter (`app/`)
- **Linter:** Uses `flutter_lints` with `package:flutter_lints/flutter.yaml`.
- **Custom Rules:** `prefer_const_constructors` is `true`, `avoid_print` is `false`.
- **Formatting:** Standard `dart format` rules apply.

## Python / FastAPI (`backend/`)
- **Style:** Standard PEP 8 conventions for Python 3.11+.
- **Typing:** Extensive use of `pydantic` models for data validation and API payloads.
